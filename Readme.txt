간단한 실행법 포함 (Readme.txt): 여러분들이 제출한 소스코드를 평가하기 위해 제 연구실의 컴퓨터에서 동작시킬 때 정상적으로 동작 시키기 위한 방법(절차)를 소개해주면 됨.

다른 동작 방법도 있겠지만 실제로 제가 실행한 방법으로 동작 방법을 설명하겠습니다.

1. 안드로이드 스튜디오를 깐다.
2. 에뮬레이터(Pixel 2 API 28) 설치 -> 다른 에뮬레이터를 사용하거나 크롬과 같은 웹 브라우저를 사용할 경우 화면에서 한글이 깨지거나 화면의 구성(위젯들의 위치)이 pixel 2 API 28로 작업한 것과 다를 수 있다.
3. visual studio code를 깐다. 
4. 확장 프로그램에서 flutter을 설치한다.(dart는 flutter을 설치하면 자동으로 설치된다.)
5. ctrl+shift+p를 눌러 명령 팔레트를 실행한 다음 Flutter: Select Divice를 선택한 다음 설치한 에뮬레이터(Pixel 2 API 28)을 선택한다.
6. 그리고 F5을 눌러 코드를 실행시킨다.
